## Процесс игры: 

1. Крутишь казик (space) (пока не готово)
2. На слотах выпадают мечи и враги
3. Если собирается комбинация из мечей, то он попадает к игроку
4. Враги спавнятся исходя из комбинаций слотов
5. Убиваешь всех врагов
6. Крутишь казик.
